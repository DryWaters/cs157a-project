ONCLUSION This paper presents a simple open 003exible computing architecture for a generic software de\002ned radio that is completely vendor independent The proposed architecture removes external dependencies like embedded processors and external memories It integrates an FPGA-based soft processor core with full host-root communication protocols vendor independent data logging architectures on the 003y software loading via custom boot loader and other logic that allows any generic FPGA device to be used as a software de\002ned radio with minimal dependencies This allows building a system that can be quickly and easily adapted to different SDR platforms FPGAs made by different vendors and with different host interfaces with minimal re-engineering without sacri\002cing the 003exibility and simplicity of the entire system For high-security applications cryptographic blocks can be integrated into the system to separate secure and non-secure data communication areas within the FPGA
