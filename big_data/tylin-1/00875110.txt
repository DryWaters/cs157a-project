generates volumes of data So applying the approach of data mining to research these data i.e web mining becomes a new promising important research field The data in the world of the web mainly include the web pages the web structures in web pages and the server log So the web mining includes web content mining web structure mining and web usage mining When a user accesses a web site he will leave his access log A medium size site can record several megabytes per day The log of a period of time is the
